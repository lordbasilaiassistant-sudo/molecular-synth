A nanofactory you can actually build

DNA origami is the most reproduced result in molecular nanotechnology: a single ~7 kb scaffold strand folded by ~200 short “staple” oligos self-assembles into an arbitrary nanoscale shape, in a single heat-and-cool pot. Molecular Synth turns that into a one-command pipeline + a desktop rig.

How a request becomes matter

shape → A-trail scaffold routing → staple-break yield optimizer (Tm balance · loop-closure · off-target · cross-dimer) → order oligos → fold (90→20 °C in Mg²⁺) → gel-verify → optional silica hardening

The synthesizer — just ask

Type a request like the Orville synthesizer. The brain (a JS port of the repo's makerlib) decomposes it, gives an honest feasibility verdict, routes it to a maker, and shows the machine plan — and declines what the synth can't make.

What it makes — today

The flagship is atomic-precision nanostructures. Two cheap macroscale makers share the same “request → compiler → machine recipe” pattern. Everything else is honestly labelled on the vision tab.

🧬 Nanostructures flagship

DNA origami shapes; rung-2 functionalisation positions enzymes/nanoparticles at chosen sites; optional silica/metal hardening. Atomic precision, nanoscale.

🧪 Drinks

“iced oat latte” → pumps + Peltier hot/cold. Assembly from stocked ingredients, not atomic synthesis.

💧 Water

Tap/reservoir (or off-grid AWG) → carbon filter → UV-C → glass. Harvest + treat, not matter synthesis.

The honest line

A universal “materialise anything from energy” replicator is north-star (E=mc²). What’s real is a federation of cheap desktop makers + an AI request brain, with the DNA nanofab as the deep-tech rung that’s genuinely novel. We add makers only where today + cheap allows.
